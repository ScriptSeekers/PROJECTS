# AIML Projects

A collection of AI and Machine Learning projects demonstrating various applications and techniques.

## Projects

### 🤖 AI Assistants & Chatbots
- **[J.A.R.V.I.S](J.A.R.V.I.S/)** - Advanced voice assistant with 150+ commands, database persistence, and system integration
- **[Chat-Bot](Chat-Bot/)** - Desktop conversational AI using Google's Gemini API
- **[CareerPath](careerpath/)** - AI-powered career guidance web application with psychometric testing
- **[cognitive analyzer](cognitive%20analyzer/)** - AI-powered cognitive behavior analysis tool for learning improvement

### 🖼️ Computer Vision & Image Processing
- **[Action Reflection](Action%20Reflection/)** - Real-time pose detection and skeleton visualization with neon effects
- **[Img Anz](Img%20Anz/)** - AI-powered image analysis using Google Gemini
- **[ObjectDetection](ObjectDetection/)** - Real-time object detection using YOLOv3

### 🏥 Health & Medical
- **[Health Diagnosis Tool](Health%20Diagnosis%20Tool/)** - Web-based health diagnostic assistant
- **[Notebook-Disease_Prediction_Using_Machine_Learning](Notebook-Disease_Prediction_Using_Machine_Learning/)** - ML disease prediction from symptoms
- **[Smart Health Monitoring](Smart%20Health%20Monitoring/)** - Health risk assessment with synthetic data

### 📊 Data Analysis & Prediction
- **[Car_Prediction](Car_Prediction/)** - Used car price prediction using Random Forest
- **[Data analysis with pandas](Data%20analysis%20with%20pandas/)** - Fundamental pandas operations on car dataset
- **[Sentiment Detection](Sentiment%20Detection/)** - NLP-based text sentiment analysis
