# Data Analysis with Pandas

A straightforward data analysis project demonstrating fundamental pandas operations on a car dataset.

## Features

- **Data Loading**: Reads CSV data from the Car_Prediction dataset
- **Statistical Analysis**: Computes averages and descriptive statistics
- **Data Quality Check**: Identifies missing values in the dataset
- **Educational Focus**: Clean, step-by-step approach for learning pandas

## Dependencies

- pandas

## How to Run

```bash
.\.venv\Scripts\python "Data analysis with pandas.py"
```

## Analysis Workflow

1. **Display Data**: Shows the first few rows using `head()`
2. **Calculate Average**: Computes mean value of 'Engine HP' column
3. **Summary Statistics**: Generates descriptive statistics with `describe()`
4. **Data Quality**: Reports missing values in each column
