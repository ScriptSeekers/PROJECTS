# ScriptSeekers Bottle Hunt 🍾
### Premium Social Experience Game

**Bottle Hunt** is a high-end web-based "Truth or Dare" game featuring a peak premium aesthetic. Designed for social gatherings, it combines elegant visuals with smooth gameplay mechanics.

## ✨ Key Features
- **Premium Glassmorphism UI:** Stunning transparent panels with mesh-animated backgrounds.
- **Dynamic Gameplay Screen:** Interactive bottle spinning and player selection.
- **Multiplayer Slot System:** Supports multiple players with custom identifiers (Arpit AV and others).
- **Responsive Design:** Optimized for both desktop and mobile devices.

## 🛠️ Technology Stack
- **HTML5:** Semantic structure with premium accessibility.
- **Vanilla CSS:** Custom design system with glassmorphism effects and animations.
- **JavaScript:** Real-time game logic, rotation physics, and modal handling.
- **Google Fonts:** Utilizing the 'Outfit' typeface for a modern look.

## 🚀 How to Play
1. Open `index.html` in your favorite browser.
2. Ensure you have the players ready in the designated slots.
3. Click the **SPIN NOW** button to initiate the hunt.
4. Prepare for the challenge!
