# Daily Oracle 🌌

A premium, standalone cosmic guidance application built with modern web technologies.

## Features
- **Deterministic Guidance**: Logic based on zodiac signs and current mood frequencies.
- **Peak Aesthetics**: Ultra-low-latency glassmorphic UI with animated cosmic backgrounds.
- **Zero Dependencies**: 100% standalone HTML/CSS/JS. No Python or external APIs required.
- **Offline Capable**: Works purely in the browser.

## How to Use
1. Open `index.html` in any modern web browser.
2. Enter your Name and Birth Date (DD / MM).
3. Select your current mood.
4. Consult the heavens for your daily path.
