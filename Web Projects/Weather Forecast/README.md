# ScriptSeekers Weather ☁️
### Hyperlocal Weather Insights

**ScriptSeekers Weather** provides real-time atmospheric data with a focus on visual immersion. Its dynamic UI adapts to provide a clear view of current conditions and future forecasts.

## ✨ Key Features
- **Real-time City Search:** Instant weather lookup for global cities.
- **Dynamic Backgrounds:** UI that changes its visual state based on location and weather.
- **Advanced Forecasts:** Detailed 7-day outlook including humidity, wind, and UV indices.
- **Micro-animations:** Subtle cloud and weather icons that breathe life into the data.

## 🛠️ Technology Stack
- **OpenWeather API:** Integration for accurate global weather data.
- **JavaScript:** Asynchronous data fetching and DOM manipulation.
- **Vanilla CSS:** Dynamic styling engine for weather-based visuals.
