# ScriptSeekers Finance 💰
### Elite Wealth & Expense Tracking

**ScriptSeekers Finance** is a professional-grade wealth management tool that provides deep insights into your financial health through beautiful visualizations and real-time tracking.

## ✨ Key Features
- **Visual Analytics:** Real-time charts powered by Chart.js for income vs. expense distribution.
- **Transaction Ledger:** Detailed history with categorized entries and balance calculations.
- **Glassmorphic Stats:** High-impact status cards with trend indicators.
- **Premium Dark UI:** Sleek, high-contrast interface designed for "Elite Wealth" tracking.

## 🛠️ Technology Stack
- **Chart.js:** Industry-standard charting library.
- **HTML/CSS:** Premium glassmorphism design language.
- **JavaScript:** Real-time data processing and state management.
