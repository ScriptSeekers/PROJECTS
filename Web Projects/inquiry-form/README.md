# ScriptSeekers Inquiry Portal 📋
### Premium Client Onboarding Interface

**ScriptSeekers Inquiry** is a professional business tool designed to capture client requirements in a high-conversion, split-layout interface.

## ✨ Key Features
- **Split-Layout Design:** Engaging brand storytelling on one side with a minimal form on the other.
- **Premium Dropdowns:** Fixed and styled select components for a seamless user experience.
- **Step-based Guidance:** Logically organized sections (Contact Info vs. Project Details).
- **Responsive Form:** Optimized inputs and buttons that adapt to any screen size.

## 🛠️ Technology Stack
- **HTML/CSS:** Premium brand-focused design system.
- **JavaScript:** Client-side form validation and submission handling.
