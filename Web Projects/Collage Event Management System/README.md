# ScriptSeekers Events 🎭
### Secure Campus Management Portal

**ScriptSeekers Events** is a multi-tier platform designed to manage and facilitate campus life. It provides specialized interfaces for Students, Event Leaders, Teachers, and the Principal.

## ✨ Role-Based Features
- **Student:** Discover events, join clubs, and track participation points.
- **Event Leader:** Create event proposals and manage attendee lists.
- **Teacher:** Review and coordinate specific events as a supervisor.
- **Principal:** High-level administrative control, approving teachers and final event permissions.

## 🛠️ Technology Stack
- **JavaScript Units:** Clean, modular JS with custom database abstraction (`db.js`).
- **CSS Architecture:** Premium design system featuring `glass` panels and `animate-fade` transitions.
- **Data Layer:** In-memory object-based storage with persistence logic simulations.
