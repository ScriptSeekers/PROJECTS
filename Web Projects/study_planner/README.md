# ScriptSeekers Academy 🎓
### AI-Powered Study Insight Platform

**ScriptSeekers Academy** is a comprehensive study planning platform that helps students organize their learning sessions and track progress using intelligent insights.

## ✨ Key Features
- **AI Study Plan Generator:** Create personalized study schedules based on your goals and style.
- **Focus Mode (Pomodoro):** Integrated timer to maximize productivity and deep work.
- **Session History:** Track your past study attempts and generated plans.
- **JWT & OTP Auth:** Secure authentication system with one-time passwords for student safety.
- **Premium Dashboard:** A high-contrast dark mode UI designed for long study sessions.

## 🛠️ Technology Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3 (Glassmorphism), Vanilla JavaScript
- **Database:** SQLite (Relational data storage)
- **Email System:** SMTP integration for OTP delivery
