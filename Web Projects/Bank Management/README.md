# ScriptSeekers Bank 🏦
### Premium Account Management Portal

**ScriptSeekers Bank** offers a minimalist and secure interface for managing multiple financial accounts. It focuses on clarity, security, and a premium "High Net Worth" aesthetic.

## ✨ Key Features
- **Account Aggregation:** Manage and view multiple accounts within a single dashboard.
- **Transaction History:** Detailed logs of deposits and withdrawals for every account.
- **Dynamic Dashboard:** Real-time balance updates and trend analysis.
- **Premium Navigation:** A sleek sidebar-based layout with "Glass-BG" visuals.

## 🛠️ Technology Stack
- **Modern HTML:** Structured for responsiveness and accessibility.
- **Premium CSS:** Custom-built components with 'Outfit' typography.
- **JavaScript:** Client-side state management for seamless navigation.
