### ✨ Design Principles
All ScriptSeekers applications follow a strict **Premium Design Language**:
- **Glassmorphism:** Elegant use of blur, transparency, and subtle borders.
- **Dynamic Backgrounds:** Animated gradients and mesh patterns that bring tiles to life.
- **Micro-interactions:** Smooth transitions, hover effects, and playful UI feedback.
- **Dark Mode First:** Optimized for eye comfort with deep, curated color palettes.
